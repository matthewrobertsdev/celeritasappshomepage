Spiffy Clock FAQ

Why is night mode not making the text dark?

For night mode to have effect, you must be using dark mode. If it is not darkening the text, you must not be using system-wide dark mode at that time.

Why is the text dark in light mode?

This is the convention on iOS devices. In light mode, the backgrounds are lighter and the text is darker. This is particularly important for colors like light mode yellow on which light text wouldn't show up very well.
